Calculation methods

  • Absolute

    Use this method if you don't yet have a reference dive with perfect weighting.

  • Relative

    Use this method if you have a reference dive with perfect weighting but wants to adapt to a new environment.

Details

  • Absolute

    With this method, Lead evaluates the amount of weight to carry based on characteristics of the diver body, the environment and the set of gears used.

  • Relative

    With this method, Lead evaluates the amount of weight to be added or removed relatively to a reference dive done by the same diver.

    It compares the diver weight, set of gears and the environment characteristics of the reference dive and the dive being planned (typical situation when travelling).
